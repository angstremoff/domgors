# DomGo

DomGo - современная платформа для поиска недвижимости.

## Структура проекта

- `/web` - Веб-приложение (React + TypeScript)
- `/android` - Android приложение (будет добавлено позже)
- `/ios` - iOS приложение (будет добавлено позже)
